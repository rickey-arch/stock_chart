# 非鉄金属全体
金：
. 激化する地政学リスク：ベネズエラ情勢と中東の緊迫
ベネズエラ情勢の緊迫化: 1月初旬、米国によるベネズエラのマドゥロ大統領の拘束・軍事介入の動きが報じられ、中南米における軍事的緊張が急拡大しました。これが「予測不能な地政学リスク」として投資家の恐怖心理（VIX指数）を煽り、無国籍資産である金への避難を加速させています。
中東・欧州の火種: イランでの反政府デモの長期化や、ロシア・ウクライナ情勢の固定化に加え、EUによる防衛強化の動きなど、世界各地で「軍事・防衛」が経済の中心テーマとなったことが、価格を一段押し上げています。
2. 米国の保護主義と「関税合戦」によるインフレ懸念
関税強化の波: 特定の輸入品に対する大幅な関税引き上げが発表されたことで、サプライチェーンが混乱。これが米国内の物価を直接的に押し上げる「輸入インフレ」を招いています。
リスク回避の買い: 貿易摩擦による経済成長の鈍化と物価上昇が同時に進むなか、株式や法定通貨（米ドル含む）への信頼が揺らぎ、価値が毀損しにくい金に資金が集中しています。
3. 中央銀行による「脱ドル化」の加速
中国やロシアをはじめとする新興国の中央銀行が、ドル依存を減らすために金の準備高を猛烈な勢いで積み増しています。
公的需要の厚み: 2025年から続くこの傾向は、2026年に入りさらに加速。中央銀行による買いは「一度買ったら簡単には売らない」長期保有が前提であるため、市場の浮動在庫を減らし、価格の下値を強力に支える構造的な要因となっています。
4. 国内要因：円安ドル高の継続（日本市場特有）
日本の投資家にとっては、国際的な金価格の上昇に加え、為替の円安進行がダブルパンチ（プラス要因）となっています。
財政不安と円安: 日本の財政健全化への懸念から円が売られやすい地合いが続いており、ドル建ての金価格が上昇しながら円安が進むことで、国内のグラム単価が驚異的なスピードで「3万円」という心理的節目を突破しました。

# 日本製鉄
判断理由：
・東京製鐵の決算発表が振るわず，場中に下落していた
・鉄鋼需要の減少・中国の鋼材輸出の増加・原材料高は業界全体の共通課題であり、日本製鉄にも飛び火すると予想した

・指標としては割安であり、下降トレンドがどこまで続くは未知数だったのでモニタリングの必要があった（実際5日目は上昇に転じた）

# ステラファーマ
判断理由：
・革新的な治験効果が権威ある論文に掲載されたとのIRが出された
・IRを受け、1/26時点で市場の上昇が始まったが、IRの重要度を踏まえるとさらなるジャンプアップの見込みは高いとみてベットした

振り返り： 
・翌日にはIRを受けて市場がしっかり反応し、さらに1/29には新たな治験契約を公表という追加材料もあったのが運が良かった
・市場の加熱傾向もなく下がる要因もないことからリスクは小さかったと考えられるため、大きくロングに出たのは妥当な判断だったと考える

# 三菱重工業・川崎重工業
判断理由：
・上昇トレンドが落ち着き、調整に入っている雰囲気
・国策の防衛予算増大が上昇の要因だったことから、解散を受けて政権交代リスクから下降トレンドに転じたと考えられ、高市政権の継続見込みは高いことから下降トレンドも一時的なものであると考えた
・直近でアナリストのレーティング引き上げ
・寄り付きで一気に上昇したことから下降トレンドの終点と考え飛びついた

振り返り：
・円高進行を受けて下降トレンドは継続した
・上昇の見込みはないと判断し、川崎重工業のショートで損切りを早めに行ったのは良い判断だったと思う
