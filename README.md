振り返り： ・実際下降傾向が継続し，ショートの判断は妥当だった ・株価としては割安であり、下降トレンドがどこまで続くは未知数だったのでモニタリングの必要があった。（実際5日目は上昇に転じた）

#ステラファーマ 判断理由： ・革新的な治験効果が権威ある論文に掲載されたとのIRが出された ・IRを受け、1/26時点で市場の上昇が始まったが、IRの重要度を踏まえるとさらなるジャンプアップの見込みは高いとみてベットした

振り返り： ・翌日にはIRを受けて市場がしっかり反応し、さらに1/29には新たな治験契約を公表という追加材料もあったのが運が良かった ・市場の加熱傾向もなく下がる要因もないことからリスクは小さかったと考えられるため、大きくロングに出たのは妥当な判断だったと考える。
